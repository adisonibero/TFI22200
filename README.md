## Requisitos
* Python 3.13.x

## Repositorio
```yml
Clonar Proyecto: git clone <project>
Descargar Cambios: git pull
Subir Cambios: git push
```

## Configuración
```yml
Entorno Virtual: python -m venv .venv
Verificar Entorno: python -m site
Generar Archivo: pip freeze > requirements.txt
```

## Entorno Virtual
```yml
Windows PS: .venv\Scripts\activate
Windows PS: .venv\Scripts\Activate.ps1
Windows CMD: .venv\Scripts\activate.bat
MacOS: source .venv/bin/activate
Linux: source .venv/bin/activate
```

## Windows Comandos
```yml
# Permitir scripts solo en la sesión actual
PowerShell: Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

## Ejecutar Ambiente
```yml
Entorno Virtual: python -m venv .venv
Windows PS: .venv\Scripts\activate
Verificar Entorno: python -m site
Requerimientos: pip install -r requirements.txt
```
